# Listening Simple UI Update

Perubahan:
- Tampilan listening dibuat lebih simpel dan modern.
- Cerita tidak lagi tampil di dalam card/kotak besar.
- Label seperti `Cerita 1`, `Cerita 2`, dan `Soal 1` dihilangkan.
- Kalimat cerita langsung tampil sebagai teks biasa dan tetap berada di halaman.
- Soal juga tampil polos tanpa card tambahan.
- Yang tetap berbentuk tombol/kotak hanya pilihan jawabannya.
- History listening tetap memanjang ke bawah dan bisa discroll.
- Setelah semua listening selesai, tombol `Selesai` tetap muncul terlebih dahulu.
